How to Play 🎨

Each player is on their own device. You'll need a voice or video call to give clues out loud.
1
The host creates a room and shares the 4-letter code. Everyone else opens this page and joins with that code.
2
Each round, one player is the Cue Giver 🎤. They get a private screen showing a secret color somewhere on the board — nobody else can see it.
3
The Cue Giver says one word or phrase out loud to describe the color — something evocative, not the color name itself. ("sunset", "avocado", "2am"…)
4
Everyone else takes turns clicking the tile they think matches the clue. You each get one guess. There's a timer — don't overthink it.
5
The secret color is revealed. Points are awarded based on how close each guess was.
Scoring
3★ Dead on
2Solid box
1Dashed box
+1Cue giver, per guess in box
